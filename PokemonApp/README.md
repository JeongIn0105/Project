📞 포켓몬 연락처 앱 만들기
네트워크 통신을 이용해서 서버에서 랜덤 포켓몬 이미지를 불러오고, 연락처를 저장하는 앱을 개발합니다.

📋 과제 설명
1. **`UIKit` 화면 구성 및 화면 전환**
    1. 화면구성: `UITableView`, `UILabel`, `UITextView`, `UIButton`  활용
    2. 화면전환: `친구 목록 페이지` → `연락처 추가 페이지`로 이동
    
2. **`URLSession` / `Alamofire` 복습**
    
    네트워크 통신을 이용해서 서버에서 랜덤 포켓몬 이미지를 불러옵니다.
    
    두 가지 방법으로 모두 개발해보면 좋은 연습이 됩니다.
    
    - 포켓몬 API: https://pokeapi.co/

1. **`ViewController 생명주기` 개념**
    
    친구 목록 페이지에 진입할 때마다 목록이 `이름순으로 정렬`되도록 합니다.
    

1. **`CoreData` / `UserDefaults` 활용**
    
    연락처 데이터를 `기기 디스크에 저장`합니다.
    
    두 가지 방법으로 모두 개발해보면 좋은 연습이 됩니다.

📱 각 주요 기능
1. 랜덤 포켓몬 이미지 불러오기 기능
사용자가 랜덤한 포켓몬 이미지를 생성할 수 있습니다. 

2. 저장한 연락처 추가 화면 수정하기 기능
저장한 연락처를 테이블 뷰의 셀을 클릭해서 들어간 뒤 이미지와 이름, 전화번호를 수정할 수 있습니다.

3. 경고 기능
이름이나 연락처를 입력하지 않거나, 이미지를 생성하지 않고 적용 버튼을 누르게 되면 Alert 메시지가 뜨도록 예외 처리하였습니다.

🤖 AI 활용
1. 레벨 5에서 적용 버튼을 클릭하면 친구 목록 화면에 저장한 이미지, 이름, 전화번호가 노출되어야 하는데 노출이 되지 않자 AI를 활용해 해결하였습니다.
   
2. 레벨 3 기능 구현하는 데에 있어 Back 하고 적용 버튼이 파란색으로 보이지 않자 AI를 활용해서 해결하였습니다.
